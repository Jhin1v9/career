# Dashboard NEXO

## Estado

Producto interno · Desarrollo activo

---

# ¿Qué es Dashboard NEXO?

Dashboard NEXO es la plataforma interna que desarrollé para centralizar el funcionamiento diario de NEXO Digital.

Conforme la empresa empezó a crecer, cada nuevo proyecto generaba más ideas, tareas, clientes, reuniones, documentos y pequeños procesos que terminaban repartidos entre distintas aplicaciones.

Llegó un momento en el que dedicar tiempo a organizar el trabajo empezaba a ser casi tan costoso como desarrollar los propios proyectos.

En lugar de adaptar nuestra forma de trabajar a varias herramientas externas, decidí construir una plataforma diseñada específicamente para nuestra empresa.

---

# El problema

Necesitábamos un lugar donde poder gestionar toda la actividad diaria.

No quería depender de una aplicación para las tareas, otra para las notas, otra para los clientes y otra diferente para automatizaciones.

Además, muchas de las herramientas existentes obligaban a cambiar nuestra forma de trabajar.

Yo quería exactamente lo contrario.

Construir una herramienta que se adaptara a nosotros.

---

# La idea

Diseñar un centro de operaciones donde toda la información importante de la empresa estuviera conectada.

No solo un gestor de tareas.

Sino una plataforma desde la que pudiéramos controlar prácticamente todo nuestro trabajo diario.

Cada nueva necesidad debía poder incorporarse sin romper el resto del sistema.

Por eso el proyecto fue creciendo como un ecosistema modular.

---

# Funcionalidades principales

Actualmente el Dashboard incluye diferentes módulos, entre ellos:

* Gestión de proyectos.
* Gestión de tareas.
* Organización de ideas.
* Seguimiento de objetivos.
* Administración de clientes.
* Control de pagos.
* Gestión de documentación.
* Sistema interno de votaciones.
* Registro de mejoras.
* Integración con Telegram.
* Integración con Discord.
* Notificaciones automáticas.
* Panel administrativo.

Cada uno de estos módulos nació para resolver un problema real que aparecía durante el trabajo diario de la empresa.

---

# Integración con Luna

Una de las características que más me interesaba desarrollar era la integración con Luna.

Muchas acciones que antes requerían navegar por distintos apartados ahora pueden realizarse mediante lenguaje natural.

Por ejemplo:

* Crear tareas.
* Registrar pagos.
* Buscar información.
* Organizar proyectos.
* Automatizar procesos internos.

Mi objetivo era reducir el número de pasos necesarios para realizar acciones habituales.

---

# Filosofía del proyecto

Nunca quise construir simplemente otro gestor de proyectos.

Mi objetivo era desarrollar una plataforma capaz de evolucionar junto con la empresa.

Cada vez que aparece una necesidad nueva, en lugar de buscar una herramienta externa me pregunto si tiene sentido incorporarla directamente al ecosistema.

Gracias a esta filosofía el Dashboard continúa creciendo de forma natural.

---

# Mi aprendizaje

Este proyecto me enseñó la importancia de pensar en sistemas completos.

No se trataba únicamente de desarrollar pantallas.

Había que diseñar relaciones entre módulos, mantener una experiencia coherente y asegurar que todas las herramientas funcionaran juntas.

También aprendí que muchas veces una buena herramienta interna puede ahorrar más tiempo que desarrollar una nueva funcionalidad para un cliente.

# Qué representa este proyecto

Dashboard NEXO representa mi forma de entender la productividad.

No consiste en utilizar muchas herramientas.

Consiste en construir las herramientas adecuadas.

Es un proyecto que continúa evolucionando junto con la empresa y que utilizamos diariamente para organizar nuestro trabajo, reducir tareas repetitivas y mantener conectado todo nuestro ecosistema digital.

Más que una aplicación, es el núcleo operativo de NEXO Digital y uno de los proyectos donde más he aprendido sobre organización, automatización y desarrollo de productos internos.
