# Proyectos Destacados

## Luna — Agente de IA autónomo

`Node.js · Express · PostgreSQL · Playwright · PM2` — github.com/Jhin1v9/luna-kernel

Sistema que permite a modelos de lenguaje ejecutar acciones reales en mi equipo a través de su propia interfaz web, sin depender de APIs de pago: detecta las llamadas a herramientas en la respuesta del modelo, las ejecuta localmente (archivos, shell, git, APIs —123 herramientas—) y devuelve el resultado al chat. Incluye bot de Telegram, CLI y sincronización entre varios equipos.

## NEXO Workspace — Portal cliente-agencia

`Next.js 14 · TypeScript · Prisma · PostgreSQL` — github.com/Jhin1v9/nexo-workspace

Plataforma de gestión cliente-equipo en monorepo (Turborepo + pnpm): 2 aplicaciones con dominio compartido (22 capabilities de negocio), control de acceso por roles, chat en tiempo real con SSE y una API REST documentada dedicada a agentes de IA. TypeScript estricto y tests end-to-end con Playwright.

## NEXO Dashboard — Sistema operativo de la empresa

`Node.js · Express · React · PostgreSQL · WebSocket`

Plataforma en producción que centraliza la operación diaria: CRM de leads, finanzas con proyección de caja, tareas con menciones y e-mail con Gmail OAuth. Integrada con Luna: el equipo crea tareas, registra pagos y consulta información por lenguaje natural desde Telegram. Unas 300 rutas de API, actualizaciones en tiempo real y 84 tests de integración.

## Landing Page Creator — Generador con IA

`Node.js · Express · Svelte · SQLite · SSE` — github.com/Jhin1v9/nexo-lp-creator

Genera landing pages completas por conversación con IA mediante un pipeline de 4 fases con streaming en vivo. Cada página puede convertirse en plantilla reutilizable: la IA elimina automáticamente los datos personales (nombres, logos, contactos) y la publica en NEXO Store. Economía virtual interna de 3 monedas y unos 300 tests.

## TPV Restauración — Ecosistema para hostelería

`React 19 · TypeScript · Supabase · PWA` — github.com/Jhin1v9/TPV-SORVETERIA-DEMO

Digitalización completa de una heladería (cliente real): TPV, sistema KDS para cocina, gestión de inventario y app instalable para el cliente, todo sincronizado en tiempo real con fallback offline. Internacionalización completa en español, catalán, portugués e inglés.

## NEXO Store — App store del ecosistema

`Next.js 16 · React 19 · TypeScript · Supabase` — github.com/Jhin1v9/nexo-digital-store

Catálogo navegable de 75 productos propios con búsqueda, filtros y páginas de detalle con demo en vivo. Sistema de valoraciones con notificación a Telegram, panel de administración protegido y recepción automática de plantillas desde Landing Page Creator.

## bug-detector-pro — Librería de bug reporting

`TypeScript · Express · SQLite · IA` — github.com/Jhin1v9/bug-detector-pro

Widget de reporte de bugs para desarrolladores: session replay, capturas anotadas, captura de consola y red, e inspector de DOM, con análisis automático mediante IA. SDK para React, Vue y JavaScript vanilla, cloud self-hosted e integraciones con GitHub Issues, Jira y Slack.
