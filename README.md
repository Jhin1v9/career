# Filosofía

No creo que un buen ingeniero sea quien memoriza más tecnologías.

Creo que un buen ingeniero es quien sabe aprender, adaptarse y encontrar soluciones.

La tecnología cambia constantemente.

La capacidad para comprender un problema y construir una solución útil permanece.

Intento mantener siempre una mentalidad de aprendizaje continuo.

Cada proyecto es una oportunidad para descubrir una forma mejor de hacer las cosas.

# Mi forma de trabajar

Cuando comienzo un proyecto intento seguir siempre una idea muy sencilla.

1. Comprender el problema.
2. Investigar diferentes soluciones.
3. Diseñar una arquitectura sencilla.
4. Construir un primer producto funcional.
5. Mejorarlo mediante iteraciones.

Prefiero evolucionar un producto constantemente antes que intentar diseñar una solución perfecta desde el primer día.
